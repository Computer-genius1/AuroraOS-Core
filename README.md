# AuroraOS-Core

**AuroraOS-Core** is the open-source foundation for a touch-first, Windows-inspired mobile OS shell.  
It provides a Start menu, example apps, and a clean SwiftUI framework for developers to experiment with OS-style interfaces and build apps on iPhone.

---

## Features included in this base code
- Core touch-first OS shell  
- Start menu with app launcher  
- Example internal apps: File Explorer and Notes  
- Swipe gestures (e.g., swipe-down to open Start)  
- Minimal SwiftUI architecture, easy to extend

---

## Premium features (not included)
- Split-screen multitasking  
- Floating windows / draggable panels  
- Virtual file system or cloud integration  
- Live tiles, polished UI themes, advanced system apps  

These features are exclusive to the official AuroraOS release.
